# evc
Implementation of Deep Mind publication 2016 - Early Visual Concept Learning with Unsupervised Deep Learning
paper address: https://arxiv.org/abs/1606.05579

There are three keypoints:
- Continuous data stream - like what the babies see.
- VAE - unsupervised learning.
- Redundancy reduction - learning statistically independent components within the data.
